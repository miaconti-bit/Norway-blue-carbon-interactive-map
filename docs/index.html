Norway Blue Carbon Map

Interactive study-site map of Norwegian seagrass (Zostera marina) and kelp (Saccharina latissima and others) sampling locations, overlaid with habitat polygons, marine protected areas, human-pressure layers, and sediment carbon data. Built as part of the UC San Diego MAS CSP capstone in collaboration with C-BLUES / NIVA (Norwegian Institute for Water Research).

Open the map GitHub repo

Site symbols

Kelp / macroalgae — orange diamond, size ∝ sample size n
Seagrass / eelgrass — purple square, size ∝ number of cores

Co-location analysis (always shown on habitat polygons)

Pressure nearby + <1% protected — pressure signal within 5 km, minimal MPA overlap
Pressure nearby + protected — pressure within 5 km and ≥10% MPA overlap
Protected habitat — ≥10% MPA overlap, no nearby pressure
Study site within 5 km — mapped habitat with a study site nearby
Evidence gap — no study site, pressure, or strong protection signal
